### Lab15 Dynamic Type: Guess Result1
จงหาผลลัพธ์ของ typeof ดังต่อไปนี้

```JavaScript
typeof 'x';         //string
typeof x;           //Number
typeof ``;          //String
typeof Infinity;    //Number
typeof(100 / 0);    //Number
typeof(1 + '3');    //String
typeof(1 * 'a');    //Number
typeof null;        //Object
```
